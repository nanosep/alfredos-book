# Alfredo's Book Web

Este proyecto define y construira una web en Astro basada en el libro `Salvese Quien Pueda`, orientada a convertir el contenido del PDF en una experiencia navegable e interactiva.

El objetivo es publicar una version HTML del libro con estructura por capitulos, bloques multimedia, calculadoras practicas por seccion y un calendario de tareas estacionales para la gestion de una finca autosuficiente.
